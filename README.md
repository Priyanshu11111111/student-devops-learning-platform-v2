# DevOps Learning Platform

A beginner-friendly full-stack DevOps project.

## Features

- Frontend UI
- Backend API using Go
- MySQL database
- Docker Compose setup
- Nginx reverse proxy
- GitHub Actions CI/CD

## Run Project

```bash
docker compose up --build
```

## Open Application

Frontend:
http://localhost:3000

Backend API:
http://localhost:8080/api/courses

Nginx:
http://localhost

## Add Courses

Use the frontend input box to add new courses.
Data is stored directly in MySQL.
